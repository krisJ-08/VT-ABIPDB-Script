Usage:

1. create virustotal account
2. obtain vt API key
3. create abuseipdb account make sure to verify account
4. obtain abuseipdb API key
5. edit the .env file nano .env 
6. in the .env file, write VT_API = "your obtained API key from virustotal"
7. append AIPDB_API = "your obtained API key from abuseIP"
8. paste the IPs in ips-input.txt (make sure that there are no spaces otherwise the result will be N/A)
9. run the program by pressing the run button
10. copy and paste the csv to excel as special text and set the delimiter to comma ","
